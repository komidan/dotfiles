### Dotfiles

Base desktop environment was XFCE on Debian 12 Bookworm Edition

PROGRAMS:
- i3          (window manager)
- polybar     (i3-status replacement)
- kitty       (terminal emulator)
- oh-my-posh  (prompt theme engine)
- nitrogen    (sets background)
- nvim >v11   (text/code editor) (can't get via apt)
